  3. também é possível adicionar links ao meu site! Tem algumas formas de fazer:
    • Podemos acrescentar links externos, que direcionam a gente para outros sites que não são o nosso.
      Pra adicionar esse tipo de link, podemos escrever uma frase, por exemplo "Clique aqui para ser direcionado ao site do Task Cursos.
      Pra isso, iremos selecionar o texto que vai virar link, apertar command + shift + P para abrir a aba de "formatação", em seguida vamos colocar o comando A, e clicar enter. Quando fizer isso, automaticamente vai ser digitado um novo texto, onde iremos inserir dentro da aspas o link.
      Porém existem algumas ressalvas, por exemplo:
      1. Fazendo apenas isso, com o link "cru", quando o cliente clicar nele, irá abrir na mesma página, o que não é recomendado para links externos.
        Pra corrigirmos isso, vamos fazer o seguinte:
        1. Dentro da abinha do comando que abriu, depois das aspas, você irá digitar TARGET, que significa o alvo, e digitar _blank. Isso vai dizer pra ele que você quer que abra uma página em branco. Em seguida, digite REL= external.

        2. Uma dica importante para o SEO, é indicar que aquele link é NOFOLLOW, pro Google entender que não precisa analisar aquele link.

Pra você inserir um link interno, que direciona pra uma página no seu site, é só fazer o primeiro procedimento.
Uma boa prática nesse caso é indicar se aquele link é anterior ou prévio.
Pra fazer isso, você vai digitar rel= PREV ou NEXT.
